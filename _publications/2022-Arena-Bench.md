We propose using Analog and Temporary On-chip Memory (ATOM) cells for on-chip training of artificial neural networks (ANNs). ATOM, with adjustable retention timescales, addresses challenges in edge device learning, demonstrating benefits in tasks like MNIST digit recognition. The controlled decay of temporary memory provides advantages akin to regularization, indicating its utility in on-chip learning before transitioning to non-volatile memories for storage and inference tasks. This suggests an algorithm-circuit codesign for effective on-chip learning of ANNs.

We propose using Analog and Temporary On-chip Memory (ATOM) cells for on-chip training of artificial neural networks (ANNs). ATOM, with adjustable retention timescales, addresses challenges in edge device learning, demonstrating benefits in tasks like MNIST digit recognition. The controlled decay of temporary memory provides advantages akin to regularization, indicating its utility in on-chip learning before transitioning to non-volatile memories for storage and inference tasks. This suggests an algorithm-circuit codesign for effective on-chip learning of ANNs.
